# FIK - CARE

**FIK - CARE** adalah sistem website untuk membuat laporan atau pengaduan terhadap Fakultas Ilmu Komputer. Sistem ini dirancang untuk memudahkan mahasiswa atau pengguna dalam melaporkan masalah terkait **Layanan** dan **Fasilitas**, serta mempermudah admin dalam menindaklanjuti laporan tersebut.

Sistem ini memiliki dua tipe pengguna:
- **User:** Dapat membuat laporan dan memantau status laporan yang dibuat.
- **Admin:** Dapat mengubah status laporan, menghapus laporan, dan mengelola pengguna yang terdaftar.

## Fitur

### Untuk User:
- Membuat laporan pengaduan.
- Melihat status laporan.

### Untuk Admin:
- Mengubah status laporan (diproses, selesai, dll.).
- Menghapus laporan pengguna.
- Menghapus pengguna terdaftar.

## Kategori Laporan
1. **Layanan** – masalah administrasi, akademik, atau pelayanan fakultas.  
2. **Fasilitas** – masalah laboratorium, ruang kelas, atau fasilitas umum lainnya.
